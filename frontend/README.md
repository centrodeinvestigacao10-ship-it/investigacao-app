# Frontend - Controle de Investigação

Estrutura inicial do app em Next.js com Tailwind.

## Scripts
- `npm run dev`
- `npm run build`
- `npm run start`

## Supabase (Backend)
1. Copie `.env.local.example` para `.env.local`.
2. Preencha `NEXT_PUBLIC_SUPABASE_URL` e `NEXT_PUBLIC_SUPABASE_ANON_KEY`.
3. Use `supabase-schema.sql` para criar as tabelas no Supabase.

## Pastas principais
- `src/app`: rotas do app
- `src/components`: componentes reutilizáveis
- `src/styles`: estilos globais
